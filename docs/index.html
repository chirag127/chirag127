chirag127 – GenAI Utility

Welcome! This project showcases a simple Python script that leverages OpenAI's API to generate intelligent text responses.

Features

  • One‑line command‑line interface.
  • Configurable model (default: gpt-3.5-turbo).
  • Secure handling of API keys via environment variables.

Get Started

git clone https://github.com/chirag127/chirag127.git
cd chirag127
pip install -r requirements.txt
python main.py "Your prompt here"
